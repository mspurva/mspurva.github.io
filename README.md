# Purva Jain's Portfolio

Welcome to the source code of my professional portfolio website. This site showcases my experience as a Business Analyst, my strategic impact, skills, and certifications.

**Live Site:** [mspurva.github.io](https://mspurva.github.io)

## Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## Components

The site is built with modular components located in `src/components`:
- **ProfileHero:** Introduction and high-level summary.
- **StrategicImpact:** Highlights of key contributions and achievements.
- **Experience:** Detailed professional journey.
- **Skills:** Categorized professional competencies.
- **CertsEducation:** Educational background and industry certifications.

## Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mspurva/mspurva.github.io.git
    cd mspurva.github.io
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to see the result.

## Build and Deployment

The site is configured for static export and hosted on GitHub Pages.

To build the project:
```bash
npm run build
```

The output will be generated in the `out/` directory (if configured for static export) or `.next/`.
